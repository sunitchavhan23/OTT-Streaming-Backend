# OTT Streaming Service Database Management System

A simple **OTT Streaming Service Database Management System** built using **SQL** and **Oracle Database** concepts.  
This project manages users, subscription plans, content, watch history, and payments for an online streaming platform.

---

## Features

- User Management
- Subscription Plan Management
- User Subscription Tracking
- Content and Genre Management
- Watch History Tracking
- Payment Management
- Sample Data Insertion

---

## Technologies Used

- SQL
- Oracle Database
- Relational Database Concepts

---

## Database Tables

### Users
Stores user account information.

### Subscription Plans
Stores available streaming plans and pricing.

### User Subscriptions
Tracks active subscriptions of users.

### Content
Stores movies and series details.

### Genres
Stores content categories.

### Content Genre
Manages many-to-many relationship between content and genres.

### Watch History
Tracks watched content history.

### Payments
Stores user payment records.

---
## Sample Functionalities

- Create and manage users
- Add subscription plans
- Track user subscriptions
- Store movies and series information
- Maintain watch history
- Record payment transactions


## How to Run

1. Install Oracle Database or Oracle SQL Developer.
2. Open the SQL file.
3. Execute all queries.
4. Verify tables and inserted sample data.
